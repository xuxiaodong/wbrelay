wbrelay
-------

wbrelay 即 Weibo Relay。它允许你方便的从命令行下发布微博。

用法
====

    wbrelay <status> [<picture>]

示例
====

要发布一条微博，可执行以下命令:

    wbrelay 'Hello wbrelay!'

如果要发布一条带图微博，则执行:

    wbrelay 'Hello wbrelay!' /path/to/picture.png
